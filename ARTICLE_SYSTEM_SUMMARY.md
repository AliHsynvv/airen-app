## 🌐 Route Structure

```
/articles                 # Article listing page
├── ?search=query        # Search functionality
├── ?category=id         # Category filtering
└── /[slug]              # Article detail page

/news                     # News listing page
├── ?search=query        # News search
└── ?sort=newest|popular # Sort options
```
